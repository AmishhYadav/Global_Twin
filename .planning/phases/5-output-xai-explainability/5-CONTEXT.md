<domain>
## Phase Boundary

This phase translates the raw mathematical arrays generated by the Simulation Engine (Phase 4) into a human-interpretable JSON payload. It wraps predictions with ± statistical error bounds and traverses the Knowledge Graph (Phase 3) backwards to narrate the causal timeline of the shock logically.
</domain>

<decisions>
## Implementation Decisions

### Confidence Intervals
- Apply a static ±RMSE confidence bound mapping onto the predicted trajectory datapoints for simplicity, guaranteeing mathematical transparency to the end-user.

### Text-based Explanations
- Develop a linked DAG traceback narrative. The explanatory string for each dependent node must recursively track its strongest influencing feature back to the shocked origin node (e.g. "Inflation (+2%) was driven by Shipping (+10%) which was driven by Oil (+20%)").

### Payload Archival
- Assemble a unified nested JSON structure containing both the explicit timeline integer arrays, bounding lists, and the text narrative strings per node.
</decisions>
